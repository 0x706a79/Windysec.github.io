0x02 Medium RSA

其中有两个文件

首先使用openssl在pybkey.pem中提取n、e

得到的是十六进制

所以

大数分解得到q、p。使用rsatools生成私钥.

python rsatool.py -p 275127860351348928173285174381581152299 -q 319576316814478949870590164193048041239 -o priv.key

使用私钥解密文件

openssl rsautl -decrypt -in flag.enc -inkey priv.key -out flag.txt
